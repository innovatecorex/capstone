SECTION EDIT — FIXED (replaces the broken section-edit.zip)
===========================================================

PROBLEM
  The earlier section-edit used @json([...]) with PHP => syntax inline in an
  onclick attribute, which Blade could not parse — causing a 500 ParseError
  on /admin/sections.

FIX
  Rebuilt from the clean original. The row data is now built in a @php block
  and passed via Js::from() (Laravel's safe JS encoder). Adds the Edit button
  + modal correctly. Editable: grade level, section name, adviser, capacity,
  status.

  php artisan view:clear
